# Assets Directory

This directory contains images and visual assets for the README and documentation.

## Expected Files

- `example_output.png` - Screenshot of example analyzer output
- `architecture.png` - Diagram showing the analyzer architecture

## Usage

These images are referenced in the main README.md file to provide visual examples of the tool's capabilities.

To add images:
1. Place PNG or JPEG files in this directory
2. Reference them in markdown files using relative paths: `![Description](assets/filename.png)`
